　　第635章
<br/>　　百里胜有些糊涂，他在京城的富家公子间很有人缘，同样政敌不少，也并不与谁有过命的交情。此时此刻乃是绝地，谁来谁送死，为何无端端有人来投靠自己？
<br/>　　但现在不是讲这些的时候，黑天半夜好一顿厮杀，先带人冲杀出去要紧，至于这卢俊义究竟如何来的，活过今天再问不迟。
<br/>　　看那卢俊义，端地是好俊的武艺，身高九尺有余，使一条金纂提炉枪，跨一匹九花大马。在战阵中厮杀，宛若一头脱闸猛虎，出海蛟龙，身周十步之内，无人是他一合之将。
<br/>　　“弟兄们，与我冲。”
<br/>　　百里胜没料得这卢俊义竟然有如此了得的武艺，只想对着穹庐狂呼一声『天不亡我』。他口中高喊一声，向外厮杀，陷入绝境的士兵人人抱着必死的决心，一时间士气如虹，锐不可当。
<br/>　　“将军，他们要突围成功了？”副将指着远处道。
<br/>　　赵子龙点点头，目光始终落在卢俊义身上，口中道：“那人却是一条好汉。”
<br/>　　百里胜拼死冲杀，竟硬生生将严密的包围圈撞出一个缺口，宋伯康也没想到会有这一幕发生，立刻下令堵住缺口，将百里胜所率领的队伍赶尽杀绝。
<br/>　　双方士兵厮杀在一起，一时间惨叫声喊杀声响成一片。
<br/>　　百里胜在卢俊义的护持下突出包围圈，宋伯康命令士兵拦截，可又有谁会是卢俊义一招之将。
<br/>　　“你们在此地按住阵脚，不要轻举妄动。”赵子龙吩咐道。
<br/>　　“属下遵命。”副将答应一声，忽然一怔：“将军，你要去做什么？”
<br/>　　却见赵子龙已经跨着战马，从草坡上冲了下去。那还听得见副将的话，只是将手中亮银枪举在空中挥了挥，算是与其打过招呼。
<br/>　　卢俊义带着百里胜向外冲杀，却见远处一小将一阵风似的向这边冲杀过来。黑天半夜，本看不太清楚，可关键是赵子龙太过显眼些。
<br/>　　他白盔白甲身披白色大氅，手持亮银枪，座下一匹白龙马，卢俊义便是想不注意他也不容易。
<br/>　　来若疾疾之风，乱军纠缠在一起，却根本无法对赵子龙形成有效阻拦。眨眼之间，他就冲到卢俊义面前。
<br/>　　“好汉留步，接我一枪。”赵子龙横枪立马。
<br/>　　“来！”
<br/>　　卢俊义虽惊不惧，挺枪向赵子龙冲来。只听铛地一声，两杆枪撞在一起，赵子龙心中一惊，此人好大力气。明知对手强大，他却没有半点怕的心思，反而激起他的好战之心。
<br/>　　抖索精神，亮银枪施展开来，百鸟朝凤枪施展到极致。一枪刺出，宛若千万条枪同时投出，空气被撕裂发出阵阵鸣声。
<br/>　　但就算他拿出全部实力，竟然也奈何卢俊义不得。对手防得滴水不漏，圆润自在，没有露出半点破绽。
<br/>　　如果一枪不够，便再来一枪，如果不够快，那便再快一些。
<br/>　　赵子龙正色起来，枪法更加凌厉。不够快，不够快……冥冥之中，赵子龙觉得自己已经摸到极限，进一步便可破开瓶颈，破而后生。
<br/>　　“走啦！”
<br/>　　百里胜口中大吼，眼前实在不是恋战的时候。
<br/>　　卢俊义狠狠啐了一声，再有片刻他想胜过赵子龙不难。但此刻还是突围更要紧，再不走宋伯康的追兵就冲过来了。
<br/>　　奋长枪击退赵子龙，抱拳道：“娃娃，今日先放你一马。”
<br/>　　赵子龙心中叹息，他虽然很想与这等高手交手一番，但他也并不想阻拦百里胜，从而变相帮助宋伯康。
<br/>　　“山高路远，日后再决个胜负。”
<br/>　　赵子龙没有纠缠，倒是出乎卢俊义预料，他临走之前看了赵子龙一眼，拍马向远处逃去。
<br/>　　宋伯康引着大军冲杀过来，口中大呼小叫：“赵将军，拦住他，拦住他。”
<br/>　　说着，他冲到赵子龙面前，道：“赵将军，你怎么不拦住他。”
<br/>　　“唉，敌人强悍，赵某拼力阻拦，不是其对手。”
<br/>　　赵子龙说得不是谎话，他的境界和卢俊义有一定差距，再打下去输的一定是他。
<br/>　　宋伯康瞥了他一眼，心道：没想到呀没想到，你这细皮嫩肉的小白脸也会扯谎了，果然，从蛤蟆城出来的没一个好东西。
<br/>　　“王上，属下愿意领一支兵马，前去阻击敌军。”戴白道。
<br/>　　“好，你速速前去，一定将他们拦住，将敌军全部歼灭。”
<br/>　　“属下遵命。”
<br/>　　戴白领着凉州城的精锐骑兵去了，宋伯康忧心忡忡，事情的变化的确出乎他预料。
<br/>　　“王上勿要担心，戴将军前去一定不会有问题的。”赵子龙道。
<br/>　　宋伯康点点头，忽然道：“赵将军，你的病没事了？”
<br/>　　“呃……这个……咳咳。”赵子龙突然捂着胸口咳嗽起来：“王上不问，我倒是忘了。刚才一心阻拦敌人，却忘了自己是带病之躯。眼前敌军主力已经被歼灭，剩下的只是些漏网之鱼。王上，赵某暂时告退，若有用得着在下的地方，请与我家大当家说话。”
<br/>　　宋伯康看了他一眼，不是很愿意搭理他。他开始组织人手，清扫战场。天快亮的时候，戴白已经领着队伍回来了。
<br/>　　“怎样，将敌人消灭了没有？”
<br/>　　宋伯康迫不及待的开口询问，但见到戴白的样子后，他就知道自己不该问。戴白丢盔弃甲，身后领着一支残兵败将，结果究竟怎样已经不问自明。
<br/>　　戴白垂着头惭愧道：“属下追上百里胜，与他打了一战，可敌军强悍，属下实在是……”
<br/>　　宋伯康冷哼一声，表情已经极其不快。不过这件事也怪不得戴白，谁也没想到百里胜在绝地竟也能突围而出。
<br/>　　“报。”一名小校着急报来：“王上，大事不好。”
<br/>　　宋伯康一激灵，自己都已现在这个样子，还能有什么情况，比现在更糟糕。
<br/>　　“发生什么事了？”
<br/>　　“那个赵将军闯入狼山关，抢了一批辎重武器，大摇大摆走了。”
<br/>　　“什么？”
<br/>　　宋伯康一惊，心中恨得咬牙切齿。这赵子龙不愧是做山贼出身，什么时候都忘不了抢劫。
<br/>　　一批兵器自然不会对宋伯康有损，关键是这件事实在令他堵心。不过他现在还不想和程大雷撕破脸，也没有这个能力，只能先将这口气忍下去。
<br/>(本章完)
