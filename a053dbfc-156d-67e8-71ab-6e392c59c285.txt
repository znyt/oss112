　　什造被安排和他住在了一起，这是刘哔主动要求的，毕竟在这样的地方安全程度并不算高。
<br/>　　经过这么长时间的耽搁，什造身上的伤势，也恢复了许多，想必很快就会从昏迷中醒来。
<br/>　　确认房间没有什么问题之后，刘哔从空间戒指中取出芳村艾特的尸体，发动了吞噬技能。
<br/>　　“叮~恭喜宿主吞噬SSS级喰种芳村艾特，血统提升，再生能力提升，获得技能幻影。”
<br/>　　随着系统声音的响起，芳村艾特的尸体大幅度缩水，很快变为了一具干尸。
<br/>　　但收获比刘哔一开始预期的要少很多，作为实力不弱于店长的喰种，芳村艾特并没有让刘哔融合太多东西。
<br/>　　或许这也和他等级提升有所关联，不过吞噬掉艾特，他也算提升到了SSS级。
<br/>　　将已经没有利用价值的尸体收起，刘哔打开技能面板，查看了一下新获得的幻影技能。
<br/>　　名称：幻影
<br/>　　类型：被动技能
<br/>　　评分：A级
<br/>　　介绍：来源于《东京喰种》位面，能大幅度提升宿主速度属性，无任何消耗。
<br/>　　只是一个A级技能，不过好在不需要消耗，还算是个不错的技能。
<br/>　　像这种纯增幅的被动技能，很少能有太高的等阶，虽然是A级，但也比一些更高等级的主动技能要珍贵许多了。
<br/>　　除了这个技能之外，还有其他几个方面的增幅，况且，借助这这次吞噬，他也成功晋级。
<br/>　　花费了这么大的功夫，这样的收获倒还可以接受。
<br/>　　关闭技能面板，刘哔犹豫了一下，查看了一下自己的积分余额，虽然之前因为救治什造而花费了不少，但剩下的还有一万多积分。
<br/>　　这些积分他并不打算动用，反正目前为止他也没什么特别需要的东西，刘哔想攒着兑换更高级的物品。
<br/>　　“这是哪里？”就在他沉入系统查看技能的时候，什造却在这个时间醒转了过来。
<br/>　　心中稍稍一动，刘哔从系统界面退出，看到已经清醒的什造，半躺在旁边的双人床上。
<br/>　　“哦，一个喰种的住所，我们暂时在这里避一下风头。”随手将外套脱掉，刘哔坐在了什造身边。
<br/>　　闻言，什造点了点头，有些好奇的打量了一下四周，并没有多说什么。
<br/>　　她自然知道，当时自己救下了刘哔，就很难再被CCG所容纳了。
<br/>　　搜查官和喰种是势不两立的，她当时的做法，估计会引起很多人的不满，甚至是怀疑。
<br/>　　作为最大的喰种收容中心，库克利亚自然不会缺乏摄像头，估计当日所有的情况，都被摄像头给录制下来了。
<br/>　　对于CCG，她并没有什么值得留念的，只是大叔那边，总要去说一声，交代一下。
<br/>　　要不，以大叔的性格，估计会很伤心的吧。
<br/>　　心中暗自琢磨着，什造打定主意，等伤势好转一些，就去见一面大叔，反正也不急于一时。
<br/>　　……
<br/>　　两人在房间里聊了一会，门外传来了一阵敲门声，刘哔止住了话头，起身打开了大门。
<br/>　　敲门的是那个紫发少年，好像叫叶·冯·罗斯华尔德，名字十分难记，对于这个喰种，刘哔的印象并不深刻。
<br/>　　唯一的印象，只是他对月山习十分忠诚，后期月山家覆灭之后，好像被改造成了特殊喰种。
<br/>　　和原著中相比，这家伙要更加女性化一些，看其细嫩的皮肤，很难想象会出现在一个男人身上。
<br/>　　“刘先生，习大人已经摆好宴席，让我来请您前去赴宴。”见到大门打开，叶恭恭敬敬的行了一礼，对两人做了一个请的手势。
<br/>　　看了一眼什造，刘哔有些犹豫，喰种的食物，在他印象中似乎不怎么适合人类。
<br/>　　“刘先生请放心，习大人为这位女士额外准备了食物，都是一些常见的菜肴。”注意到刘哔的眼神，叶面带笑容，礼貌的解释了一声。
<br/>　　点了点头，这次他也就没再拒绝，在叶的带领下来到了餐厅。
<br/>　　四周古色古香的装饰，很难让人看出这是喰种栖息的地方，虽然对这方面没有太多的研究，可刘哔依旧能看出，周围的这些家具都价值不菲。
<br/>　　餐厅中央，一个长桌前，上面摆满了精美的菜肴，琳琅满目，大部分是肉食，也有一些面包蔬菜之类的食物。
<br/>　　月山习坐在长桌首座，身边站着两三个仆人。
<br/>　　“刘先生，请坐，希望您能对这次晚宴感到满意。”月山习站起身来，脸上带着优雅的微笑，打了个招呼。
<br/>　　回了一礼，刘哔拉着什造，随意找了个位置坐下，面对一桌的食物，却没有动筷。
<br/>　　他不太知道岛国用餐之前的礼节，但至少也清楚，主人未动之前，客人是不能先吃的。
<br/>　　而月山习，似乎也没有什么吃饭的意思，看到刘哔入座，月山习举起手中的高脚杯，仆人很懂事的给两人倒上了红酒。
<br/>　　“刘先生，之前的事情金木已经和我说了，真是没想到，您居然是高贵的SSS级喰种。”双手举着酒杯，月山习站起身来：“这杯酒，请让我给您赔罪。”
<br/>　　暗自皱了皱眉，刘哔脸上挂起有些虚伪的笑容，客气的和月山习碰了碰杯子，放在唇边抿了一口。
<br/>　　微甜，如同丝绸一般的润滑口感，还有一股掩盖不住的血腥味道，杯子里盛放的，是血液。
<br/>　　怪不得仆人没给什造倒上，人类可不会喝这种东西的。
<br/>　　“这是我珍藏很久的幼童血液，里面稍微加了一些酒精，用特殊的手法存放了十几年，方才制作成功的，刘先生，感觉口感如何？”
<br/>　　端着高脚杯，月山习满脸陶醉的表情，不得不说，这样的血液的确非常美味。
<br/>　　“月山习，不用和我绕圈子了，有什么事情，直接说就好了。”刘哔放下酒杯，浓郁的血腥味让他双眼已经开始发红，似乎有一口火焰盘旋在心里，让他有些压抑。
<br/>　　因为血腥味的刺激，他感觉自己的身体都躁动了起来。
<br/>　　察觉到刘哔的状态有些不对，什造握住了刘哔的右手，冰凉凉的，让他心中那股火焰稍稍减弱了一些。
