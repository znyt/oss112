　　第309章
<br/>　　对于草原上的情形，要问过樊梨花才清楚。云中龙几个都没来过草原，至于福德勒，虽然是戎族，但北蛮部距离这里十万八千里远，也是睁眼瞎。
<br/>　　据樊梨花所说，这片草原上分布着十几个部落，有大有小，最大的要数西戎部。
<br/>　　戎族没有文字，没有货币，甚至没有所谓的权力结构，如此，也不怪帝国视他们为蛮夷。帝国的铜钱他们是不认的，当然金银什么时候都是硬通货，只不过作价太低。
<br/>　　这里的人更习惯以货易货，还处于商业的最初级阶段。
<br/>　　樊梨花告诉程大雷，这些部落随着水草迁移，想要找到他们也不容易，不过，在这片草原，有一个固定的交易地点，算是一个市场，各部落每月定期在此交易所需物资。
<br/>　　交易的货物包括牛羊马匹，猎犬，兵器，铁器，茶叶，当然也有贩卖的奴隶。
<br/>　　算起来明天就是交易的日子，程大雷将大部队留下，自己和樊梨花、福德勒三人各背了一些货物前往交易市场，也算是提前探探路。
<br/>　　路上，程大雷又问了一些关于戎族的风土人情，樊梨花还称得上了解，可福德勒就少言寡语了。
<br/>　　这个孩子心里究竟想什么，谁也摸不清楚，有时候问上三句，都未必会回答一句，程大雷也不是很愿意和他聊。
<br/>　　三人骑了三匹马，马上驮着货物，药材、茶叶、盐都带了一些。
<br/>　　在黄昏时分，赶到了市场。这是位于草原上的一片红树林，明天才是正式开市的日子。但陆陆续续已经来了不少人，程大雷看到红树林中，一伙人一伙人窝在一起，面前堆着货物。
<br/>　　银器、玉石、草药、成群的羊羔，凶恶的猎犬等等之类，也无人招呼，谁有需要的，就自己上前商议价格。
<br/>　　当程大雷三人出现时，各人都投来不善的目光，像盯着猎物的豺狼。
<br/>　　以帝国和戎族的深仇大恨，他们没有当场拔刀已经是很友善了。程大雷心里不禁有些打鼓，这要是所有人都冲上来，己方三人只能麻溜快的逃走。
<br/>　　樊梨花却是表现得很坦然，她真的像是来逛街购物一样，随意看着。
<br/>　　“戎族崇拜强者，只要你比他们强大，他们就尊重你。什么仇呀恨呀，对他们来说并不重要。”樊梨花随意说道：“各个部落间也经常发生战争，仔细算起来每家都有杀父之仇，夺妻之恨，也算不清楚。只要你比他们强，哪怕抢了他们的老婆，他们依旧愿意做你的奴隶。”
<br/>　　说着，樊梨花看了程大雷一眼：“当然，这种事你是做不来的。”
<br/>　　樊梨花小小的嘲讽了程大雷一下，程大雷打了个哈哈。
<br/>　　“那是，那是，我们是文明人么。”
<br/>　　听到樊梨花的话，程大雷不再有什么心理负担。也陪着樊梨花逛这市场，有人满脸凶恶的看着他，程大雷也凶恶的看回去，最后目光对峙，对方识趣的避开。
<br/>　　哈，这还真是文化不同，习俗也不同。
<br/>　　樊梨花说得不错，戎族的确崇拜强者，各部落间互相抢女人的事情更是屡见不鲜。在这个地方，只要你拳头够大，本事够强，就不缺乏有人奉你为王，哪怕你杀了他老父，睡了他老母。
<br/>　　市场上贩卖的稀奇古怪的货物不少，某种奇怪动物的皮毛，从河滩捡到的石头，也有押着十三四岁女童贩卖的，而有些东西，程大雷更是见所未见，闻所未闻。
<br/>　　程大雷眼睛一亮，忽注意到一件物事，单筒，黄铜质地，上锈古怪花纹。
<br/>　　这应该是一个欧式望远镜。
<br/>　　卖货的是一个戎族汉子，相比常见的戎族，他身躯比较瘦弱。盯着程大雷的眼神像只凶恶的老鼠。程大雷却也不理他，让福德勒向他询价。
<br/>　　最后，用一包盐就将这单筒望远镜买下，在草原上，盐巴是可以当货币用的。像老鼠一样的戎族汉子目光立刻便柔和起来，皱纹堆起来，像是要笑。
<br/>　　程大雷透过望远镜看着远处，其实效果并不好，说是千里镜，但怎么可能看千里。程大雷目力还算可以，这千里镜也发挥不了太大作用。
<br/>　　真正让程大雷上心的是这千里镜的来处，帝国十三州东临沧海，西北是戎族，那么再向西有什么？
<br/>　　程大雷转过镜筒，看到西方群山起伏，天高云远。
<br/>　　帝国的地形并不和程大雷前生完全相同，至于整个世界的构造，程大雷也是一无所知。世界当不止帝国，也当不止戎族。
<br/>　　西方是什么？
<br/>　　镜筒内出现一张吹弹可破雪润玉滑的俏脸，立得太近，连对方皮肤下的血管都看得一清二楚。
<br/>　　樊梨花颦起眉头：“看什么呢，看得清楚么？”
<br/>　　“啊，看得清，看得清。”
<br/>　　程大雷将千里镜收起来，挥挥手道：“来，来，把我们的货物也摆出来。”
<br/>　　三人这次只带了少量的药材、茶叶和盐巴。三个包袱打开，程大雷蹲在大树下，颇有种前世撂地摊的感觉。
<br/>　　本来，这里的戎族看三人的目光都不善，随时要动手杀了三人的意思。暂且没有动手，也只是摸不清三人的来路。
<br/>　　直到程大雷将货物摆出来，周围人的眼睛明显亮了，有人握住了兵器。程大雷也把斧头摆了上去，果然周围贪婪的目光黯淡下来。
<br/>　　能用这样沉重一柄斧头当武器的人可能是善茬么，而且那斧头上还有明显的血痕。
<br/>　　果然，很快就有第一个人过来，询问程大雷想换什么。
<br/>　　在这个时代，交易的情形大概是这样的，先问清楚对方需要什么，再用自己的东西给对方交换，假若自己没有的话，就需要先从别处交换。
<br/>　　毕竟是商业的最原始阶段，而像盐、茶叶、药材则属于硬通货，完全可以当货币使用。
<br/>　　程大雷面前围着的人越来越多，很快程大雷就体会到了戎族的豪爽热情，用一包盐换一匹骏马的事情也出现了。
<br/>　　程大雷更中意的还是美石美玉这样易于携带的，而在帝国这些东西也比较好出手。有些戎族特产的草药，也同样是程大雷所需要的。
<br/>　　
<br/>(本章完)
