　　第353章
<br/>　　暗夜里，繁星漫天。高高的露台上，垂手立着两个老者。寒露浸湿他们的衣物，他们恍然未决，依旧望着头顶星空。
<br/>　　长安，观星台。
<br/>　　紫微星在北，是无数颗星辰中最明亮的一颗，在地上人心中，这是天子的象征。
<br/>　　“咳咳咳”
<br/>　　随着一名黑袍老者的咳嗽，紫微星的光芒似乎暗淡了几分。
<br/>　　一身着僧袍的老者合十，道：“陛下，天寒露重，您要保重身体。”
<br/>　　黑袍老人摆摆手，似乎想要说什么，可突如其来的咳嗽止住了他的话，天上紫微星的光芒似乎更加暗淡。
<br/>　　“你……看出了什么？”黑袍老人问。
<br/>　　僧人叹了口气，轻唤了声佛号。
<br/>　　“贫僧不敢说。”
<br/>　　“说！”
<br/>　　“陛下，您来看！”
<br/>　　僧人的手指过去，只见天空上一颗本来暗淡的星辰渐渐明亮起来，似乎连紫微星的光芒也被夺去几分。两颗星辰在天空中对峙，已成龙争虎斗之相。
<br/>　　“哪里是什么地方？”黑袍老者问。
<br/>　　“西北。”
<br/>　　“西北是程大雷在的地方吧。”黑袍老者轻轻道了一句，却不知心中想着些什么。
<br/>　　“陛下，天命虽有数，人定却可胜天。”
<br/>　　这声音落地，突然见天域角落中，又有一颗星辰明亮起来，周天的星辉都无法阻挡它的光芒。越来越亮，忽然化作一团白色火球，在天空中一闪而逝。
<br/>　　“哪里又是什么地方？”
<br/>　　僧人脸上也露出一丝困惑，呐呐道：“哪里好像是幽州。”
<br/>　　……
<br/>　　幽州地，青牛山。
<br/>　　大山匍匐如野兽，山脉走势若龙蛇。
<br/>　　当明帝在长安露台观星时，此地山顶也有几人昂首仰望天上星辰。
<br/>　　一行七人，为首之人身后背一柄长枪，伫立山顶如旗帜。在他左右还立着两个道人，一老一少。
<br/>　　“大战过后，幽州十室九空，大地空留累累白骨。而北蛮部也是元气大伤，最后谁也没在这场战场中讨到好处。”老道悠悠说着。
<br/>　　如果程大雷在这里，当一眼就能认出来，这老道正是曾经见过的郭烦人。
<br/>　　“可这也正好是我们的机会，如今的幽州已经成了无主之地，正好应我们起事。”林少羽缓缓道。
<br/>　　假若程大雷在这里，则有可能认不出林少羽，因为他身上的变化太大了。虽然体型依旧瘦削，可整个人却变得更沉，更稳，眼里若含着星辰，不怒自威。
<br/>　　“神将大人准备好了？”郭烦人问。
<br/>　　林少羽扭过头，嘴角抿起一抹笑意，如此才有几分少年模样。
<br/>　　“分别前，大当家送了我一件东西，虽然我不知道那是什么，可自从那一日后，我渐渐变明悟了很多东西。直到最近，有些东西我才终于参透。”
<br/>　　说着，一声长啸，背上长枪握在手中，猛地发力，长枪如白龙般打在崖边一颗石头上。随着声音，磨盘大小的巨石从中央裂开，化作齑粉砸落悬崖。
<br/>　　郭烦人、红尘了、林冲几个惊愕的张大嘴巴，久久难以合拢。
<br/>　　常人总说千斤力，万斤力，终究是夸大其词的成分居多。真如果有千钧之力，那该是天上的神，地上的鬼，反正不该是人间之物。
<br/>　　可林少羽刚刚这一枪，当真有几分夺鬼神气魄的味道。
<br/>　　就连林冲也暗暗心惊，他不得不承认一个事实，目前的林少羽，已经是自己无法胜过。
<br/>　　此刻，头顶一颗星辰突然明亮起来，夺走漫天星斗光辉。
<br/>　　郭烦人突然匍匐在地，口中呼喝：“天神降世，光耀长夜。”
<br/>　　声音在山谷间传开，然后是漫山遍野的回声，山峰下一个个人跪倒，口中呼的都是八个字。
<br/>　　“天神降世，光耀长夜。”
<br/>　　在林少羽所立的山坡上，密密麻麻跪倒一个个人影，若看得仔细，就当发现，他们背上的图案一模一样。
<br/>　　一只模样丑陋，且狰狞的蛤蟆。
<br/>　　在林少羽头顶，一柄战旗飘扬，上书斗大一个『義』字。
<br/>　　……
<br/>　　“大当家，凉州那边来人了。”徐神机。
<br/>　　程大雷正坐在摇椅上打盹，无可奈何的被徐神机唤醒。他不耐烦道：“来就来了么，有什么大不了的。”
<br/>　　“带着礼物来的，大当家你不去见见么？”
<br/>　　程大雷想了想，无奈的站起身：“走喽，就去见见他们吧。”
<br/>　　来人在将军府的大厅，程大雷看到大厅的空地上摆着诸多礼物，绸缎、猪肉之类的。
<br/>　　程大雷去的时候，一个中年胖子迎出来，笑眯眯道：“这位就是牛将军吧，某凉州钟伟虎见过牛将军，牛将军果然和传言得一样，相貌堂堂，一表人才。”
<br/>　　程大雷摸了摸鼻子，虽然你说我相貌堂堂这话不假，可『牛将军』三个字从何说起。
<br/>　　他恍然醒悟，帝国封的是牛三斤，而不是自己。钟伟虎不是不知道对面是谁，只是不想捅破这层窗户纸。
<br/>　　还真够麻烦的，程大雷摆摆手：“里面请吧。”
<br/>　　诸人坐在大厅内，差仆人上了茶，钟伟虎未语先笑。
<br/>　　“自从牛将军上任，便该差人拜访，一拖二拖，实在事务繁多，便拖到了现在。”
<br/>　　帝国文官客套这一套，程大雷当真有些受不了，你开门见山，咱们直接谈赎金不好么。可对方如此客气，程大雷也不好撕破脸皮，只好虚伪的应承着。
<br/>　　“听犬子说，宋公子被牛将军留下做客，哈哈，都是青年才俊，定有说不完的话，可宋大人念子心切……”
<br/>　　砰！
<br/>　　程大雷突然重重一敲桌子，钟伟虎吓了一跳，难道自己是哪句话说错了，惹得程大雷不满。
<br/>　　“程……牛将军，是有什么问题么？”
<br/>　　“喔，没问题没问题，我突然想起来还有件急事，钟先生先在此地稍等片刻，我去去就来。”
<br/>　　说着，程大雷走出了大厅，离开钟伟虎后，程大雷感觉空气都清新许多。
<br/>　　“大当家，怎么不聊了，我看你聊得很开心么？”
<br/>　　“放和珅。”
<br/>　　程大雷算是想明白了，专业的事情还是要交给专业的人办，自己瞎凑什么热闹。
<br/>　　钟伟虎等在大厅，已有几分口干舌燥，虽然面前就摆着茶，但他却是一口都没喝。
<br/>　　正这时候，一人走进了大厅，满脸堆笑。钟伟虎忽地觉得，气氛开始变得不一样了。
