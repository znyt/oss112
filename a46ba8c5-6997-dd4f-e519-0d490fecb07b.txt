　　第211章
<br/>　　长道漫漫，随着咯吱咯吱的响声，一辆马车缓缓行来。
<br/>　　赶车的是一壮汉，身穿锈盔破甲，佩一柄木刀，这样的天气，热汗哗哗往下流，可他仍板正着身子，身体绷得像张弓。
<br/>　　“他爹，咱还有多久到长安啊？铁儿都饿了。”
<br/>　　“快啦，快啦。”
<br/>　　“走这么久，你该不会又走错路了吧。”
<br/>　　“什么叫又，我又没来过长安，是那砍柴的给我们指错路好不好。”
<br/>　　“唉，我不知道上辈子造了什么孽，才倒霉跟了你，要吃没吃，要穿没穿，唉，我要是不嫁给你就好了，我不嫁给你就不会生出铁儿，我如果不生出铁儿……”
<br/>　　“好啦，你这娘们烦不烦，到了长安就好了，这次朝廷调我上京，以后日子就好过了。”
<br/>　　“哼，你这人就是扫把星转世，跟着你一辈子倒霉。”
<br/>　　牛三斤，西北云水关守将。一百二十年前，大武开国的时候，分封功臣。这牛三斤的祖上曾经给太祖牵过马，于是被封到云水关，世袭罔替，祖祖辈辈镇守此地。传到牛三斤这里的时候，已经是第五代。
<br/>　　一百二十年，沧海桑田，足够令很多事情发生改变。云水关也不是太重要的地方，原先镇守此地的也不过一千兵丁，时过境迁，死的死逃的逃，如今就剩下七八户人家。
<br/>　　与其说牛三斤是云水关的守将，倒不如说他是云水村的村长。戎族时常犯边，却很少来云水关，主要是这个地方实在太穷了，穷得戎族都忘了这里还有人口。关键是不仅戎族把他们忘了，帝国这边也把他们忘了。
<br/>　　牛三斤祖辈每天放放羊，种种田，那日子过得叫一个惨。有时候牛三斤也会想，自己究竟还是不是帝国的武将。
<br/>　　直到去年，戎族大举入侵，把帝国摁在地下打，仗打完了，朝廷总要查一查，哪座城被戎族打破了，哪座城的将领有功。
<br/>　　终于，朝廷从泛黄的故纸堆中，找到了云水关的存档。大家惊讶的发现，在西北一线全面崩溃的情况下，竟然还有个云水关一兵未损，关键是也没有什么好损的，戎族根本没有理他们这茬。
<br/>　　明帝手捧着档案痛哭流涕，世代镇守云水关，多少年不要粮，不要钱，当然要了也没人给，任劳任怨，无怨无悔，这真是帝国的顶梁柱。
<br/>　　老祖宗就是会看人呐。
<br/>　　太多的好处，牛三斤捞不到，战功都被几大势力均分。这次正好朝廷要开武科举，那就让这个人当个考官吧，反正他不属于任何派系。
<br/>　　于是朝廷的圣旨传到云水关，牛三斤有些懵，但想来这应该是好事。于是他带着老婆孩子，穿上祖先穿过，已经有一百年历史的盔甲，没有兵器怎么办，削柄木刀当兵器吧。京城是天子脚下，总不能丢了体面。
<br/>　　“小牛啊，这次去京城，要好好表现，不要丢了咱们云水关的面子。”
<br/>　　“替我转告武帝陛下，说我老王一直念着他呢。”
<br/>　　“大叔，武帝陛下已经过世很久了。”
<br/>　　“小牛，你是咱们全村人的希望，不要丢人，在长安奔个好前程。”
<br/>　　于是，全村人的希望牛三斤，便赶着马车，带着老婆孩子，以及准备在长安城打点的草药，玉石，兽皮之类的东西，穿着锈盔破甲，佩着木刀，一路舟车劳顿，前往长安奔前程。
<br/>　　“呔！”
<br/>　　已快到长安城，路上突然跃出一捎人马，手持钢刀，黑布蒙头，为首之人开口大喊。
<br/>　　“呔，此山是我开，此树是我栽，要想从此过，留下买路财，牙嘣半个不，管杀不管埋。”
<br/>　　车上的女人吓了一跳，忙抱紧孩子：“他爹，咱们是不是遇到强盗了，唉，我就知道，当初就不该嫁给你，我若不嫁给你……”
<br/>　　“莫慌，有我在。”牛三斤一跃从车上跳下来：“尔等强人，可知道我是什么人，竟然连帝国的武官也敢劫，呀呀呀，吃我一刀。”
<br/>　　锵啷一声……抱歉，没有锵啷一声，牛三斤拔出木刀，指着强人一伙。
<br/>　　这一伙强人自然就是程大雷等人乔装改扮的，看到这一幕，程大雷等人都愣住了。
<br/>　　见过穷的，没见过这么穷的，连一套装备都凑不齐，当官也不该当得这么磕碜么。
<br/>　　“你是牛三斤？”
<br/>　　“哼，既然知道俺的姓名，还不快快退下。”
<br/>　　“当真是？”
<br/>　　“老子行不更名坐不改姓，云水关牛三斤就是我。”
<br/>　　“绑了，绑了。”
<br/>　　程大雷挥挥手，只听耳边叮当作响，牛三斤呼喝出声。
<br/>　　“呔，吃我一记旋风连斩。”
<br/>　　“看我秋风刀。”
<br/>　　“呔，你们放开我，放开我！”
<br/>　　没多时，牛三斤一家三口就被绑成粽子相仿，手下人将他身上的官凭印信搜了出来。
<br/>　　程大雷一边拿着官凭，一边看着牛三斤的相貌。
<br/>　　“方脸紫面，阔口长须。”程大雷撇撇嘴：“你这不像呐，你明明是黄脸膛，哪来的紫面，你该不会是哪个冒充的吧？”
<br/>　　“那是我爷爷的官凭。”牛三斤怒吼一声：“天大地大，谁又敢冒充俺老牛，你们想要什么冲我来，放开我老婆孩子。”
<br/>　　帝国官员的档案管理其实很落后，有时候被人冒充了也没人知道，而这种事简直可以说是层不不穷。
<br/>　　“带走，带走。”
<br/>　　大家一起动手，将三人装进麻袋里，悄悄运进地下城软禁起来。
<br/>　　而程大雷则扒掉牛三斤的破盔锈甲，攥住那柄木刀，摇身一变就成了代表全村人希望的牛三斤。
<br/>　　倒也不必完全按照牛三斤的模样乔传改扮，反正长安城也没人见过他，程大雷乔装的主要目的，是避免被人认出自己。
<br/>　　乔传改扮之后，程大雷揣上刀，带着徐神机和徐灵儿，赶着马车去官府报道。他二人一个冒充老奴，一个冒充丫鬟，反正长安城这么大，也没有人能够认得出他们。
<br/>　　程大雷却是不知道，刚到负责考官报到的衙门，就有人指着他大声道。
<br/>　　“哎呦呦，真没想到啊，你这么穷还带的起仆人呐。”
<br/>
