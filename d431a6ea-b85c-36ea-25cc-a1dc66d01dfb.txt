　　临安市，
<br/>　　一家高档餐厅里，
<br/>　　叶尘、庄菲菲和红姐，以及庄菲菲的私人律师，四人坐在包厢里，已经等了将近半个小时。
<br/>　　原本，
<br/>　　庄菲菲与华艺天皇娱乐的一位叫高总监的人约好，在下午两点在此处见面，商讨一下这次的违约之事，可是这个高总监似乎在故意摆架子，居然迟迟没有前来。
<br/>　　砰！
<br/>　　叶尘将手中的茶杯往桌子上一放，眼眸中露出一抹不耐，直接冲一旁的庄菲菲道：
<br/>　　“庄小姐，这华艺天皇娱乐公司的背后，是何方神圣？”
<br/>　　没等庄菲菲开口，坐在她身旁的那名美女律师道：
<br/>　　“华艺天皇的背后，是杨氏集团！”
<br/>　　“杨氏集团这两年发展势头很猛，据说是搭上了某位大人物，如今集团的实力，在整个东江省也能排进前三！”
<br/>　　“连带着华艺天皇的地位，也跟着水涨船高，如今已经是娱乐圈的四大巨头之一！”
<br/>　　叶尘听到这话，不由得眉头微微一挑，
<br/>　　“杨氏集团？难道是杨之凡的老子杨国强？这倒是巧了！”
<br/>　　那位美女律师听到这话，顿时双眸微微一亮，
<br/>　　“不错！杨氏集团的掌舵人，正是杨国强！他们家的确有一位公子，名叫杨之凡，据说已经确定是杨氏集团的接班人，莫非小叶先生你认识？”
<br/>　　庄菲菲并没有告诉这位美女律师，叶尘的真实身份。
<br/>　　因而，虽然她觉得这个少年，气质和谈吐都颇为不凡，应该不是一个寻常人物，多半是某个家族的公子哥，不过毕竟太年轻了，只怕也帮不上什么忙，心中难免有些轻视。
<br/>　　如今听到少年说这话，顿时来了精神。
<br/>　　华艺天皇是娱乐圈的巨头，即便是庄菲菲这样的一线明星，在它的面前，也不过就是一只蝼蚁罢了，根本没有多少反抗的余地。
<br/>　　倘若能搭上杨氏集团的太子爷，争取双方达成和解，那此事就还有回转的余地。
<br/>　　叶尘淡淡一笑，实话实话道：
<br/>　　“杨之凡是我的大学同学……”
<br/>　　“太好了！”
<br/>　　美女律师听到这话，顿时大喜过望，
<br/>　　然而还没等她高兴起来，叶尘接下来一句话，瞬间将她从云端打入深渊。
<br/>　　“不过，我和他并没有什么交情！”
<br/>　　美女律师听到这话，都差点想要骂人了，
<br/>　　既然没交情，那你说个屁啊！
<br/>　　叶尘直接无视美女律师的反应，继续道：
<br/>　　“但是，我可以让杨国强亲自来一趟！”
<br/>　　美女律师顿时两眼一瞪，露出一脸宛如看智障的表情，几乎是呼喊了起来，
<br/>　　“让杨国强亲自来一趟？你以为你是谁？那杨国强在全国富豪榜上，也可以排名前五十，是东江省最顶级的商业巨头！”
<br/>　　叶尘完全无视美女律师的质问，直接掏出手机，拨通了楚飞烟的电话。
<br/>　　很快，
<br/>　　电话那头传来楚飞烟声音，
<br/>　　“叶尘，你回云州了？”
<br/>　　“还没，我现在在临安市，遇到了一些事情，杨氏集团的杨国强，你知道吗？”
<br/>　　楚飞烟虽然很好奇，叶尘为什么会忽然问起这个，不过当即还是老老实实回答道：
<br/>　　“当然知道！杨氏集团是我们新开发的渠道商，负责紫金仙泉在东江省的销售，你问他做什么？”
<br/>　　叶尘听完楚飞烟的报告，点了点头，淡淡道：
<br/>　　“我眼下有一位朋友，遇到了一点麻烦，你通知杨国强，我现在在临安市钱潮路红玫瑰餐厅，让他半个小时之内，出现在我的面前！”
<br/>　　挂完电话之后，叶尘继续端起桌前的茶杯，开始慢悠悠的品茶起来。
<br/>　　一旁的那位美女律师，将一切看在眼中，先是一脸惊愕，继而是疑惑，再然后便忍不住嗤笑了起来，小声嘀咕了一句，
<br/>　　“小小年轻，脸皮也忒厚了！说大话也不怕闪了舌头！你要是真能一句话就能把杨国强喊过来，老娘以后就跟你姓！”
<br/>　　就连一旁的庄菲菲，也不由得露出一抹狐疑之色，
<br/>　　虽然她知道叶尘的身份不简单，可是这杨国强也不是寻常人物啊！
<br/>　　叶尘将众人的神情看在眼中，淡淡一笑，继续低头品茶。
<br/>　　又等了大约十来分钟，包厢的房门忽然被人推开，
<br/>　　一名身穿肥胖的中年男子，挺着浑圆的大肚子，笑着从外面走了进来，身后还跟着几名身材魁梧的壮汉，个个气势凛然，显然都是练家子，
<br/>　　“哈哈哈！菲菲小姐，实在是不好意思！路上有点事情耽搁了，让你们久等了，实在是抱歉！”
<br/>　　很显然，这个肥胖中年男子，应该就是庄菲菲口中的那位高总监了。
<br/>　　这货嘴上说着不好意思，脸上却尽是得意之色，而且在望向庄菲菲的时候，眼眸深处还透着一抹婬光！
<br/>　　看到此人进来，庄菲菲的经纪人红姐，和那位美女律师当即便站起了，恭恭敬敬喊了一声，
<br/>　　“高总监！”
<br/>　　庄菲菲虽然眼眸中露出一抹厌恶之色，不过最终也起身相迎。
<br/>　　唯独叶尘依旧端坐在那里，静静的品着茶。
<br/>　　高总监看到这一幕，顿时眉头一皱，两眼在叶尘身上上下打量了片刻，阴测测道：
<br/>　　“这位小兄弟倒是面生的很呐，不知道是何方神圣？”
<br/>　　叶尘这才缓缓抬起头，淡淡瞥了一眼高总监，
<br/>　　“我是谁，你还没有资格知道！我今天来，只为庄小姐讨回一个公道！”
<br/>　　哗！
<br/>　　叶尘这话一出口，众人顿时尽皆哗然，
<br/>　　就连庄菲菲也没有想到，叶尘竟然如此干脆直接。
<br/>　　那位美女律师，更是脸色大变，
<br/>　　她之前分析过庄菲菲和华艺天皇的合约，形势对庄菲菲极为不利，这个也是娱乐圈的行业惯例。
<br/>　　倘若双方撕破脸皮，对于庄菲菲来说，将会陷入极为艰难的境地。
<br/>　　因而她的谈判策略是和解。
<br/>　　然而，
<br/>　　她万万没有想到，叶尘一上来就这么牛气哄哄，直接就把氛围搞的如此僵硬。
<br/>　　果不其然，高总监听到叶尘这话，原来脸上还洋溢着的笑容，瞬间便消失的无影无踪，继而浮现出一抹寒色，冷声道：
<br/>　　“菲菲小姐，你这是什么意思？竟然找这么一个愣头青来闹事，你这是不打算和公司和谈了吗？”
<br/>(本章完)
