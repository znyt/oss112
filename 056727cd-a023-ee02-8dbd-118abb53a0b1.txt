　　萧逸自问，自己已经不是当年的菜鸟了。
<br/>　　先不说各郡皆有裂天剑派。
<br/>　　单单裂天剑宗，若真的是全炎武王国的武道圣地，为何自己从来不曾听闻。
<br/>　　“裂天剑宗？”萧逸眉头皱得更紧。
<br/>　　大长老似是早就料到萧逸有这样的表情，笑道，“我和十一，便曾是裂天剑宗的弟子。”
<br/>　　“至于你不知道，不，准确来说，是整个北山郡都不知道。”
<br/>　　“因为，我们裂天剑派和北山郡王，多年前便已经联手将这些信息封锁了。”
<br/>　　“除了那些大势力的大人物还会闲聊这些事外，没人再会提起。”
<br/>　　“为什么要封锁？”萧逸问道。
<br/>　　大长老脸上，涌出一丝愤懑和怨气。
<br/>　　“因为，我们北山郡裂天剑派，乃是各郡剑派里，最弱的一个。”
<br/>　　“其它分派，根本看不起我们，甚至孤立我们，已经许多年没有互相来往了。”
<br/>　　“连同着北山郡，也被视作了三十六郡中最弱的一郡，沦为笑柄。”
<br/>　　“当年，我和郡王一怒之下，直接将这些消息都封锁了，断了和他们的关系。”
<br/>　　“他们既然看不起我们，我们也没必要求着他们各分派理会我们。”
<br/>　　“最弱的一个？”萧逸皱着眉。
<br/>　　他想起了之前的炼药大比。
<br/>　　就拿炼药一道来说，北山郡年轻一代，叶铭几乎是最强的。
<br/>　　可是，和其它郡的炼药天才相比，却是如此不堪，根本比不过。
<br/>　　他还想起来周龙的一句话。
<br/>　　‘你们北山郡的裂天剑派如此无用，可以早早闭门解散了。’
<br/>　　现在想起来，周龙此话的意思，是指北山郡的裂天剑派，比不过其它郡的裂天剑派。
<br/>　　“裂天剑派，到底发生了什么？”萧逸沉声问道。
<br/>　　“我曾经见过其它郡的天才，真论资质，也并非卓绝。”
<br/>　　“为何我们剑派和其它分派的差距，会如此之大？”
<br/>　　“唉。”大长老重重地叹了口气，道，“还不是极界碑惹的祸。”
<br/>　　大长老忽然沉默了起来，似在思索。
<br/>　　萧逸也没有出言打扰，静静地等着。
<br/>　　半晌，大长老才继续说道，“各郡裂天剑派弟子，要进入裂天剑宗，只有一个要求。”
<br/>　　“便是参悟门派传承石碑，达一成以上。”
<br/>　　“……”
<br/>　　大长老娓娓道来，萧逸大致也明白了过来。
<br/>　　所谓门派传承石碑，便是剑派内最强的一块武道石碑。
<br/>　　本来，各分派的传承石碑，都只是元界碑。
<br/>　　都是各分派曾经出过的天元境武者前辈所留。
<br/>　　唯独北山郡裂天剑派，出了裂天剑魔这么个妖孽，留下了一座极界碑。
<br/>　　武道境界，凡境、后天、先天、洞玄、破玄、地元、天元、地极、天极。
<br/>　　虽说天元和地极，只差了一个大境界。
<br/>　　但实则，地极，是跨入极境的武者，与天元相比，这又是另外一个范畴的存在。
<br/>　　所以，参悟元界碑，一般的天才都能做到。
<br/>　　而参悟极界碑，非剑道天才，无法参悟。
<br/>　　而且就是剑道天才，其难度也极大。
<br/>　　其它各郡裂天剑派，每一代，都有弟子，参悟十成元界碑，成为剑主。
<br/>　　每一代的内门十位首席弟子，都能参悟一成以上，进入裂天剑宗。
<br/>　　唯独北山郡裂天剑派，数百年来，没有出过剑主。
<br/>　　每一代，都只有寥寥两三人能参悟极界碑一成以上，进入裂天剑宗。
<br/>　　到了大长老和易老这一代。
<br/>　　更是只有他们两人能够前往。
<br/>　　这些进入剑宗的弟子，老了之后，可选择留在剑宗；或者回到自己郡的剑派，担任掌教或者长老。
<br/>　　可想而知，各郡，每一代都有大批出色的强大武者回归剑派，教导弟子。
<br/>　　唯独北山郡裂天剑派，一代不如一代。
<br/>　　简单来说，老的都不强，如何教导小的。
<br/>　　这时，大长老也唏嘘地叹起了气。
<br/>　　“其它各郡的裂天剑派长老，最起码都是地元境，武道知识远超我们。”
<br/>　　“而我们北山郡裂天剑派的长老，除了我是地元境外，其它都是破玄境，如何比得过人家。”
<br/>　　“其实叶铭、苗千火、玉如龙他们的资质，一点儿都不差。”
<br/>　　“只是…哎，是我们这些长老没用。”
<br/>　　顺带一提，大长老和易老，是年轻时就成功参悟了极界碑，前往裂天剑宗。
<br/>　　而剑堂长老、三长老等人，是后来老了，修为大涨，才稍微领悟了那么一些，并无资格进入剑宗。
<br/>　　“据我所知，大长老参悟极界碑的成数是三成。”萧逸问道，“那么易老呢？”
<br/>　　萧逸曾经试探过易老，知道易老绝对参悟了极界碑五成以上。
<br/>　　但具体是多少成，却不知道。
<br/>　　大长老面露窘态，道，“进入剑宗前，我也不过是参悟了一成；后来从剑宗回来剑派，修为大涨，才最终参悟了三成。”
<br/>　　“反倒是十一。”
<br/>　　大长老忽然笑了，“进入剑宗前，已经参悟了七成。”
<br/>　　“他那时告诉我，其实他能完全参悟透，只不过他不想。”
<br/>　　“年轻时的十一，狂妄得很。”
<br/>　　“他说，他有自己的剑道。极界碑上的霸道剑道，他参悟七成足矣。”
<br/>　　“再参悟下去，会影响自己的剑道，便不再参悟了。”
<br/>　　“呵。”萧逸淡淡一笑，他能想象到，当年的易老，炼药一道、剑道，堪称双绝，必然是个傲气非凡之人。
<br/>　　“以易老的本事，教导弟子，不比其他郡的剑派长老差吧？”萧逸问道。
<br/>　　“关键就是在这里。”大长老忽然收敛了笑容。
<br/>　　“这也是我今天要跟你说这些事的原因之一。”
<br/>　　“当年的十一，哪怕进了裂天剑宗，也是风云人物。”
<br/>　　“在剑宗内修炼十余年，已是可独当一面的强大武者。”
<br/>　　“后来，他收了个徒弟。”
<br/>　　“这个徒弟，天资卓绝，天赋非凡，十一极为喜爱。”
<br/>　　“只可惜，后来这个孽徒…哎，不说了。”
<br/>　　“总之，十一因此性情大变，更离开了剑宗，之后四处游历。”
<br/>　　“足足漂泊了五年，才再次回到北山郡以及剑派。”
<br/>　　“回来之后，当年狂傲非凡的十一，变得沉默寡言。不再收弟子，顶多就是有空指导一番。”
<br/>　　换言之，真正教导弟子的，还是大长老还有其它长老。
<br/>　　萧逸万万没想到，北山郡的裂天剑派，竟然有着这么多变故。
<br/>　　三十六郡，最弱的北山郡。
<br/>　　三十六分派，最弱的裂天剑派。
<br/>　　极界碑，竟是祸不是福。
<br/>　　生生让得裂天剑派一代不如一代，能去裂天剑宗的弟子，更是越来越少。
<br/>　　更是让得大长老和北山郡王，联手封锁了消息。
<br/>　　免得‘最弱’二字，成为北山郡武者头上的阴影。
<br/>　　“现在，你知道我为何要偏帮顾长空了？”大长老说道。
<br/>　　“整个剑派，就他一个参悟了一成极界碑。”
<br/>　　“也只有他，有资格前往裂天剑宗。”
<br/>　　“若你死了，他也死了，裂天剑派这一代将没有人能进入剑宗。”
<br/>　　“假以时日，我们这些老家伙撒手人寰，北山郡裂天剑派就完了。”
<br/>　　萧逸点点头。
<br/>　　长老们，自然是以剑派利益为重的。
<br/>　　拿大长老来说，当年就他和易老进入剑宗修炼了十数年，彼此的情谊，必然是比其他长老要深的。
<br/>　　生生气跑了易老，大长老也很不好受。
<br/>　　“嗯？”萧逸忽然想起了什么，连忙问道，“大长老似乎还未告诉我，易老的下落，以及今日跟我说这些的原因。”
<br/>　　大长老回答道，“十一离开前，并未告诉我去了哪里。”
<br/>　　“但据我猜测，他肯定去了王都，他在那里有不少至交好友。”
<br/>　　“而我今天之所以跟你说这么多，就是因为半年后，就是裂天剑宗，开宗收徒之日。”
<br/>　　“半年后？”萧逸明白了。
<br/>　　难怪裂天剑派全派大比提前了，原来当初易老所说，更重要的事，就是裂天剑宗开宗收徒之日。
<br/>　　大长老继续道，“你要决定是否前往裂天剑宗。”
<br/>　　“因为，十一那个孽徒，如今就在裂天剑宗，而且还不是普通弟子。”
<br/>　　“另外，十一在王都，朋友多，仇人也多。”
<br/>　　“你是否要去，必须考虑清楚。你身为他的弟子，去那里可没好日子过。”
<br/>　　“不必考虑，裂天剑宗，我去定了。”萧逸脸色坚毅，毫不犹豫地说道。
