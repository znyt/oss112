   <br/>    陈兴正看着钥匙上的图案，忽然头皮传来轻微的刺痛，顺着感觉的方向望去，他对上了一双布满血丝的眼睛。<br/>    <br/>    “小子，你死定了！”<br/>    <br/>    陈兴眯了眯眼睛，没想到在这里遇上了前黑鹫佣兵团的团长琼斯·金。对方龇牙咧嘴，朝他比了比中指。<br/>    <br/>    一时间，陈兴感到头疼无比。他和琼斯·金之间没有任何妥协的余地，只有一方死亡才会结束。而且从对方衣袖上的标志看，已经是巨猿团的核心成员，十分棘手。<br/>    <br/>    却在这时，铁诺转头看了过来，随即虎目圆睁，盯着琼斯·金。<br/>    <br/>    琼斯·金面露不屑，朝地上吐了口痰。<br/>    <br/>    刹那间，铁诺身形暴涨，以肉眼可见的速度变高变大，正是他的天赋能力“躯体巨大化”。只见他大手一伸，越过长桌，一把抓住琼斯·金的衣领，拽了过来。<br/>    <br/>    骤变突生，所有人都望过来，有的更是拔出了武器。<br/>    <br/>    <br/>
