<br/>    七日之后。
<br/>    
<br/>        丁况站在高塔之顶，遥望四周，却是没有发现任何动静。这几日以来，也没有什么特殊的事情发生，一切都如常。
<br/>    
<br/>        “看来是我多虑了，并没有什么意外发生，老夫在这里待了几日，也没有发现什么异常，若是再不去鬼门星，这群废物可无法夺下鬼门星。”
<br/>    
<br/>        步留痕正在房间之中休息，这个时候，步留痕忽然听到了有天仙前来禀报，“执刀人大人，丁况这只老狐狸，根本没走，之前走的，只是他的车架，直到今天傍晚，他才真正离开。”
<br/>    
<br/>        步留痕忽然感受到了无面者的召唤，急忙闭上双眼，再睁开双眼，已经是在陈明大殿之中坐好。
<br/>    
<br/>        这是无面者面具特有的效果，可以将别人的意识带到此处。
<br/>    
<br/>        陈明看了一眼步留痕，只有一句话，“三天之后，动手。”
<br/>    
<br/>        步留痕心中一惊，无面者阁下，到底是一位什么样的存在？丁况不过前脚刚走，无面者阁下就已经知道了，而且在七天之前，无面者阁下又是怎么知道丁况压根没走的？
<br/>    
<br/>        若不是无面者阁下的提醒，自己恐怕早就已经落入了丁况的圈套了！
<br/>    
<br/>        这一切都是秘，不过他知道，不该他知道的，他不该问。
<br/>    
<br/>        “是，无面者阁下，属下一定完成任务。”
<br/>    
<br/>        陈明挥挥手，步留痕消失在大殿之中，“现在，可以看看我这柄新的刀，到底有多快了。”
<br/>    
<br/>        三日后，深夜。
<br/>    
<br/>        步留痕和十个天仙，人人带着面具，浑身黑袍，步留痕并没有戴太多的人来，他们是来救人的，不是来开战的，救人之后要迅速撤走，人多只是拖累。
<br/>    
<br/>        步留痕扫了一眼，防卫并不是很森严，两个天仙，四十位地仙。
<br/>    
<br/>        要夺下这里，轻而易举。
<br/>    
<br/>        步留痕郑重地接过了黑色匣子，将其背在了身后。
<br/>    
<br/>        步留痕向前行走，守卫很快发现了他们，“你们是什么人？此地，乃是瑶池的大狱，找死不成？”
<br/>    
<br/>        “速速止步！”
<br/>    
<br/>        两人的喊声迅速惊动了其他的守卫，齐齐涌了出来！
<br/>    
<br/>        大门打开，一个个地仙朝着外面走了出来，典狱长也是大步出来。
<br/>    
<br/>        步留痕扫了这几人一眼，挥手道：“杀了！我带着英魂兵器，不能动手！我所有的动作，都会被视作玷污了英魂兵器。”
<br/>    
<br/>        典狱长听到英魂兵器几个字，瞬间脑子里就炸开了，这些到底是什么人，他们又是来救谁的？
<br/>    
<br/>        他们带着的，是英魂兵器？
<br/>    
<br/>        难道此处的大牢之中，关押着一个能得到英魂兵器认主的大仙不成？
<br/>    
<br/>        十个天仙一掠而出，与对方的守卫战作一团。
<br/>    
<br/>        刀枪剑雨之中，步留痕走过一柄柄兵器的剑锋，踏着剑锋走进大牢之中，不过刚刚走进大牢，便是看到了一个独腿的老者。
<br/>    
<br/>        老杀才也是听到了外面的惊变，而后不久，就有一个带着面具的黑衣人背着一个木匣子来到了此处。
<br/>    
<br/>        步留痕一脚踏碎了牢门，看向老杀才，“无面者阁下，说的应该就是你吧？”
<br/>    
<br/>        步留痕将木匣子放在老杀才的身前，老杀才双眼之中微微颤抖，他有些不敢置信，他伸手，打开木匣子，一柄血红色的大弓沉睡其中。
<br/>    
<br/>        “是它，真的是它，没有想到，他居然真的得到了这柄弓.......”
<br/>    
<br/>        老杀才知道能够让这柄弓认主意味着什么，这个无面者，和之前的六个无面者都不一样。
<br/>    
<br/>        他想要伸手抓住这柄弓，却又不敢，他看向步留痕，问道：“有刀吗？”
<br/>    
<br/>        步留痕疑惑了，这老家伙想什么呢？
<br/>    
<br/>        老杀才继续道：“是无面者阁下的加价，我冒犯了无面者阁下，所以有加价。”
<br/>    
<br/>        步留痕这才想起来，差点忘了，他伸手从储物戒中取出一柄刀，丢到老杀才的面前。
<br/>    
<br/>        老杀才握刀，“好久，好久都没有握住过兵器了啊，还真是让人沉迷的滋味啊！”
<br/>    
<br/>        然后他抬手挥刀，一刀落下，斩在自己仅剩的独腿上，鲜血淋漓，流淌了一地。
<br/>    
<br/>        不过他却是如同没有感觉一般，再来一刀，终于是将自己的腿斩断。
<br/>    
<br/>        步留痕怒视老杀才，喝问道
<br/>    
<br/>        ：“你这是做什么？”
<br/>    
<br/>        老杀才呵呵一笑，“这便是无面者阁下的加价，我第一次见到无面者阁下的时候轻视了无面者阁下，无面者阁下要了我的一条腿。”
<br/>    
<br/>        步留痕的心中一震，无面者阁下，还真是手段森寒狠辣啊，还好自己没有什么冒犯的举动.......
<br/>    
<br/>        老杀才道：“将这些被囚禁的犯人放出来吧？”
<br/>    
<br/>        步留痕道：“将这些重犯都放出来！”
<br/>    
<br/>        十位天仙开始一一打开这些牢门，从中走出一个个凶神恶煞的家伙，“哈哈哈哈，多谢兄弟们高抬贵手，救出我等，为了表示感谢，我决定抢了你们，不杀你们！”
<br/>    
<br/>        这人浑身威势一震，解除了枷锁的他可是一位天仙，而且不是天庭的那种天仙。
<br/>    
<br/>        又有一个犯人哈哈大笑道：“不不不，我的规矩，杀了再抢！”
<br/>    
<br/>        还有人道：“呵呵呵，人肉，我可是馋了好久了........”
<br/>    
<br/>        “这一次出去，先去抢几个姑娘快活快活......”
<br/>    
<br/>        步留痕压根没有怎么在意，他觉得这些都是老杀才的手下，“你这些手下，还真是嚣张跋扈啊。”
<br/>    
<br/>        老杀才看向步留痕，摇摇头道：“我不认识他们啊，都是一些穷凶极恶之徒。”
<br/>    
<br/>        步留痕看着已经放出来的三十多个天仙，再看了自己这边的人手，“那你把这些穷凶极恶的家伙放出来干嘛？”
<br/>    
<br/>        老杀才呵呵一笑，“现在不认识，马上就认识了。”
<br/>    
<br/>        三十多个穷凶极恶的天仙朝着步留痕和老杀才走来，这个时候，却是听见了老杀才的声音，“无尽黑暗之中的伟大存在啊，您的执刀人在此刻呼唤您，请降临在我的身边，让我可以拥有执掌天狼弓的权柄吧......”
<br/>    
<br/>        “这老东西嘀咕啥呢？”
<br/>    
<br/>        “似乎是在请求什么，不过，他的呼唤，好像是在召唤无面者......”
<br/>    
<br/>        “什么，无面者降临？”
<br/>    
<br/>        三十多个天仙步步后退，并不敢有别的动作。
<br/>    
<br/>        只是一个名字，就足以让他们胆寒。
