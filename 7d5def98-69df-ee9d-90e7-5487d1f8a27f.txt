　　第674章
<br/>　　心怀利器，杀心自起。
<br/>　　生在大富大贵之家，这个道理崔白玉还是很明白的。宋伯康以前还算是老实，安心做相府的马仔。当然，他不安心也是无用，相府有很多办法限制他。但如今他已经是凉州王，如果没有程大雷，他已是这片土地上的土皇帝。
<br/>　　摆在他这个位置上，很难说他会安份。换做从前，他绝不敢半夜来访，顾左右而言他，目的昭然若揭。
<br/>　　当真是胆子越来越大了，却也忘了，他是如何走到今天的。相府既然能抬举他成为凉州之王，也自然能踩得下去他。
<br/>　　崔白玉不动声色，心中杀心已起。等结束这件事，下一个料理的便是宋伯康。那么，就且容这条老狗多活几天吧。
<br/>　　送走宋伯康之后，崔白玉仍旧回到书房翻书，目前，程大雷才是她真正的敌人。
<br/>　　手中翻着书，心思难免飘到别的地方去。生得太过美貌，这是她为数不多的烦恼之一。身边诸多人中，有幸窥见她美貌的人并没有几个，但无论是谁，是怎样的辈分，看见自己时，都会露出那种丑陋的眼神。
<br/>　　有些人故意装作浑不在意，但眼神总会不自觉的往自己脸上瞟过。这令崔白玉对男人这种生物很难产生好感。她甚至曾经想过，是否毁掉自己的容貌，能令自己的本事更得重用。
<br/>　　当然，这个念头只是想一想而已，毕竟，她窥见自己的相貌时，也会有几分不忍，暴遣天物，是会有报应的。
<br/>　　咚！
<br/>　　这时候西南天空忽然亮起一束烟花，在夜空中绽开后，绚然灿烂，点亮了半边天空。
<br/>　　窗户推开半扇，崔白玉窥见这一幕，下意识皱起眉头。
<br/>　　是发生什么事了么？
<br/>　　紧跟着，响起轰隆隆的爆炸声，即便崔白玉这里，地面似乎都在轻轻摇晃。
<br/>　　“发生什么事了？”崔白玉问。
<br/>　　歌舒离急忙忙过来，道：“启禀郡主，已经派人去查，相信马上就能查清楚。”
<br/>　　崔白玉点点头，静静等着结果。
<br/>　　这时候，西南方向，先以烟花为信号，潜入凉州城的白元飞纠集城中的探子，已经点燃了火药，火药炸塌了半幅城墙。
<br/>　　谁都知道，这场战斗打的就是时间差，必须争分夺秒。强行渡河的软包、麻袋、木筏丢入城中，秦蛮率领的蛮字军已经踩着木筏，冲杀进城墙。入城之后，他们第一件事就是攻占西门。
<br/>　　城中的守军根本没来得及反应，秦蛮打了他们一个措手不及。没有多久时间，西门便被占领，程大雷统率着剩余队伍一路冲杀进来。
<br/>　　他骑着黑牛，手持鬼面斧，身后还跟着举大棒的银眸。二人有一个特点，便是力气很大，手中的武器挥开，当真是无人能敌。
<br/>　　破开城门的事，比想象中更加顺利。事实上，程大雷还是太过低估自己，秦蛮这样万人敌的悍将，放在别的诸侯手中，都是手握军权的元帅。而如秦蛮这样实力的人，程大雷手中还当真不缺。
<br/>　　就连程大雷自己，也是一名绝世高手。
<br/>　　如今这个时代的战斗，一员悍勇的大将的确能影响很多事情，首先便是敌军的士气。
<br/>　　程大雷、银眸、秦蛮、赵子龙、高飞虎、高飞豹、史文恭、小白狼这八人最少也是顶级高手，悍不可挡，轻而易举便闯入凉州城。
<br/>　　至于进城后要做的事情，程大雷出发前已做好安排。他率领人去攻占城主府，赵子龙与史文恭攻占其他城门，高飞虎三人在城中搞事情，目的是将城中的守军赶走。如今凉州城大概驻军五万，真如果这些队伍醒过神来，自己这三千人当真还不够给他们填牙缝。
<br/>　　重中之重，便是占领城主府，杀了宋伯康，凉州城便是群龙无首的局面。
<br/>　　另外，程大雷还有一些私心。那白玉美人也在城主府，关于她的相貌，程大雷还是蛮好奇的。
<br/>　　城主府内，歌舒离跪倒在崔白玉面前，道：“郡主，大事不好，程大雷杀进城了。”
<br/>　　“什么！”
<br/>　　饶是崔白玉，此刻也是一惊。她抬起头，见四面火光冲天，喊杀声远远传到这里。
<br/>　　“他们闯进来多少人？”
<br/>　　“属下不知。”
<br/>　　崔白玉眉头皱的更紧，望着远处的火光，身子忽然晃了三晃，幸亏小犬扶住她，才没有跌倒在地上。
<br/>　　“我……错了。”
<br/>　　崔白玉呐呐叹了一声，没人理解她话中的意思。
<br/>　　她或许的确错了，早知道程大雷是一个不按常理出牌的人，可自己还是想得太天真。以为程大雷会给自己十天时间，在开战之前，互相打声招呼，然后便是你来我往，阴谋诡计……
<br/>　　然而，程大雷根本没有与她坐下来对弈的打算，开局便掀翻了桌子，拎着棋盘劈头盖脸的砸过来。崔白玉以为这是一场智力竞赛，可程大雷以为这是一场拳击赛。
<br/>　　“歌舒离。”
<br/>　　“属下在。”
<br/>　　“命令你的人好生护卫，我们走。”
<br/>　　“走？”歌舒离一怔。所谓的走就是逃跑，自家小姐怎样人物，怎么可以轻易逃跑。
<br/>　　崔白玉摇了摇头，程大雷夜袭凉州城，兵力一定不多，只要组织起守军，便可将其击退，或者消灭。然而，自己手中没有握着兵权，城中兵将也不会听自己指挥。
<br/>　　而宋伯康……崔白玉并不觉得他有这种能力。
<br/>　　如果是自己，闯破城门后，第一个要占领的一定是城主府。如果现在不走，怕就真的走不成了。
<br/>　　或者说，从程大雷闯破城门的那一刻，胜负已经决定。凉州的所有权属于谁，也已经决定。
<br/>　　可是她心中真的不服，因为自己的智谋指挥勇气等等能力，在这场战斗中根本没机会显现出来。
<br/>　　程大雷未必比自己更擅长战争，他只是比自己更清晰战争的本质，于是简单直接，直捣黄龙。
<br/>　　崔白玉想走，但她也并不是想走就走的。程大雷率领着大军杀到城主府，首先就将城主府撞破了。
<br/>　　手中的大斧头抡开，口中高声喝道：“宋伯康在哪里，宋伯康在哪里……”
<br/>(本章完)
